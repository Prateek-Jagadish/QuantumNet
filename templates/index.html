How QuantumNet Works

Quantum Key Distribution

Our implementation of the BB84 protocol allows Alice and Bob to generate a shared secret key through quantum communication, with Eve's eavesdropping attempts being detectable through quantum mechanics.

Secure Communication

Messages are encrypted using AES-256 with quantum-generated keys, ensuring that even if intercepted, the content remains secure. Keys automatically expire for enhanced security.

AI Security Monitoring

Our RandomForest classifier continuously monitors communication patterns, detecting anomalies, eavesdropping attempts, and potential security threats in real-time.

Real-time Web Interface

Modern web interface built with Flask and SocketIO provides real-time secure messaging, security monitoring, and user management with responsive Bootstrap 5 design.
